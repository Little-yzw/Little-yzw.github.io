从秦岭脚下到代码世界——一名西科IT人的成长之路

Posted on 2025-02-05 | In 个人随笔
西安科技大学电子信息专业毕业生，一年互联网全栈开发经验，记录从校园到职场的转型之路与技术思考


临潼校区图书馆前的那棵梧桐，见证了无数个敲代码的深夜

一、技术之路的起点：西科岁月

2017-2021年就读于西安科技大学电子信息专业，在信号处理与嵌入式系统课程设计中，第一次用Python实现了基于小波变换的测井信号去噪算法，这段经历为我后来的工业软件开发埋下伏笔。2021年毕业后加入陕西太合智能钻探有限公司，开启了我的全栈开发之旅。

二、工业软件攻坚：Python/Qt测井系统开发

项目背景：传统测井数据分析依赖国外商业软件，存在数据兼容性差、定制功能缺失等问题

技术架构：

  • 采用PyQt5框架构建GUI界面，实现多文档视图架构
  • 开发数据解析模块，支持LAS2.0/3.0、DLIS等测井标准格式
  • 基于Matplotlib实现动态曲线绘制，支持200+测井曲线的同屏渲染
  • 集成机器学习模块（Scikit-learn），实现自动异常值检测

技术突破：

  1. 研发基于OpenGL的GPU加速渲染引擎，将大规模测井曲线绘制效率提升400%
  2. 设计智能缓存机制，支持20GB+测井数据文件的流畅操作
  3. 实现与Modbus协议的实时通信模块，可直接对接井下仪器

三、内部网站系统开发：检验检测后台管理系统

项目背景：本项目是为陕西太合智能钻探有限公司开发的检验检测后台管理系统，旨在实现检验检测业务的全流程数字化管理。系统采用前后端分离架构，后端基于 Spring Boot + MyBatis Plus，前端基于 Vue 3 + Element Plus，实现了从检测任务分配到报告生成的全流程管理。

技术栈
后端技术栈

  • 核心框架：Spring Boot 2.7.x
  • ORM 框架：MyBatis Plus
  • 数据库：MySQL 8.0
  • 缓存：Redis（用于高频数据缓存和分布式锁）
  • 消息队列：RabbitMQ（用于异步任务处理）
  • 文件存储：MinIO（用于检测报告和附件存储）
  • 权限控制：Spring Security + JWT
  • 接口文档：Swagger 3.0
  • 部署：Docker + Jenkins

前端技术栈

  • 核心框架：Vue 3
  • UI 组件库：Element Plus
  • 状态管理：Vuex
  • 路由管理：Vue Router
  • 数据可视化：ECharts
  • 构建工具：Vite

功能模块
1. 用户管理

  • 功能描述：实现用户信息的增删改查、角色分配、权限管理。
  • 技术实现：
    • 基于 RBAC（角色基于访问控制）模型设计权限系统。
    • 使用 JWT 实现无状态登录认证。
    • 集成 Spring Security 实现接口级别的权限控制。

2. 检测任务管理

  • 功能描述：实现检测任务的创建、分配、进度跟踪和结果上传。
  • 技术实现：
    • 使用 MyBatis Plus 动态 SQL 实现多条件查询。
    • 集成 RabbitMQ 实现任务分配和状态更新的异步通知。
    • 使用 Redis 缓存高频查询的任务状态数据。

3. 检测报告管理

  • 功能描述：实现检测报告的生成、审核、下载和归档。
  • 技术实现：
    • 基于 Apache POI 实现动态报告模板生成。
    • 使用 MinIO 存储生成的报告文件。
    • 集成 Elasticsearch 实现报告全文检索。

4. 仪器设备管理

  • 功能描述：实现仪器设备的登记、校准、维护和状态监控。
  • 技术实现：
    • 使用 WebSocket 实现仪器实时数据采集。
    • 基于 Quartz 实现设备校准提醒任务调度。

5. 数据统计与可视化

  • 功能描述：实现检测数据的多维度统计分析和可视化展示。
  • 技术实现：
    • 使用 ECharts 实现数据可视化。
    • 基于 MyBatis Plus 的聚合函数实现数据统计。

核心代码示例
1. 后端代码（Spring Boot + MyBatis Plus）

// 检测任务服务层实现
@Service
public class InspectionTaskServiceImpl extends ServiceImpl<InspectionTaskMapper, InspectionTask> implements InspectionTaskService {

    @Autowired
    private RabbitTemplate rabbitTemplate;

    @Override
    public void createTask(InspectionTask task) {
        // 保存任务到数据库
        save(task);
        // 发送任务分配消息到消息队列
        rabbitTemplate.convertAndSend("taskExchange", "task.routingKey", task);
    }
}

// 检测任务控制器
@RestController
@RequestMapping("/api/task")
public class InspectionTaskController {

    @Autowired
    private InspectionTaskService taskService;

    @PostMapping("/create")
    public ResponseEntity<String> createTask(@RequestBody InspectionTask task) {
        taskService.createTask(task);
        return ResponseEntity.ok("任务创建成功");
    }
}
